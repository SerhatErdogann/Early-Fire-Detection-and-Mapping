Problem & Motivation

Why we built this system

Ecological Damage

Wildfires spread rapidly and cause severe ecological and economic damage, especially when detection and response are delayed.

Continuous Monitoring

Continuous aerial monitoring using drones can support faster and more dynamic wildfire detection across vast forest regions.

Multimodal Analysis

Relying solely on RGB vision is prone to false alarms (e.g. sunset, smoke). Combining RGB with Thermal cameras improves reliability.

Fire Detection Pipeline

How the AI processes real-time data

1. Inputs

Real-time RGB images, Thermal images, and GPS coordinates are captured by the drone.

2. Preprocessing

Frame extraction, resizing to 224x224, and data normalization for neural network compatibility.

3. Feature Extraction

Dual-branch Convolutional Neural Networks (ResNet) process RGB and Thermal data independently to extract critical features.

4. Fuel Scorer & Fusion

Visual features are concatenated with environmental data (NDVI, NDMI, Land Cover) fetched dynamically from Google Earth Engine.

5. Output

The system predicts Risk Probability, Fire Status, and outputs a Risk Classification sent instantly to the database.

GIS Visualization Dashboard

The backend writes predictions to a PostGIS database instantly. The Leaflet.js powered frontend visualizes the drone's flight path and fire risk zones in real-time.

  • Real-time Heatmap generation
  • Interactive risk markers (Low, Medium, High)
  • Google Maps coordinate integration
  • Dynamic mode switching (Heatmap/Points)
